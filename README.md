# 🏓 Pickleball Master

Learn pickleball rules, master strategies, and play with friends in real-time 2D games.

## ✨ Features

- **Learn Rules** — Complete guide to pickleball rules and scoring
- **Master Strategies** — Pro tips and tactics
- **Play Online** — Create or join games with up to 4 players
- **2D Interactive Game** — Real-time gameplay with physics
- **Mobile Friendly** — Touch controls + responsive design

## 🚀 Getting Started

```bash
npm install
npm run dev
```

## 🛠️ Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS

## 📱 How to Play

1. Create a game or join with code
2. 2-4 players can join
3. Desktop: Arrow keys + Space to swing
4. Mobile: Touch controls on screen
5. First to 11 points wins!
